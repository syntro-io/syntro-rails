Welcome to TestCaseDB

The homepage is your intro into what is happening in TestCaseDB. The two main components are displayed; Recent Activity and current tasks.

Recent Activity

The recent activity feed shows you recent changes within the system to keep you up-to-date.

Tasks

The tasks listing on the homepage shows tasks that are assigned to you with their due date. If the task is highlighted red, it is overdue.

Getting Started

If you're new to TestCaseDB, the dashboard won't have anything too interesting. Have your admin configure Products and Categories on the Admin page to get started. Once that done you can start creating test cases and test plans.
